# LingoVille

English learning town app built with React, Vite, Express, and Capacitor.

See [API_SETUP.md](API_SETUP.md) for AI provider, database, server, and mobile packaging notes.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Optional: set `GROQ_API_KEY` in `.env` for AI generation
3. Run the app:
   `npm run dev`
